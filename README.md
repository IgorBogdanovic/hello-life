# Hello Life website

The website is static.

## Languages used:
- HTML
- SASS (compiled to CSS)
- JavaScript

## Plugins used: 
- jQuery
- [Slick](http://kenwheeler.github.io/slick/)
- [ScrollMagic](http://scrollmagic.io/)
- [GSAP](https://greensock.com/gsap)
- Picturefill

## Tools used: 
- Node.js
- Gulp
